# Source Selection Criteria

This project includes a curated set of public references supporting strategic mindset, critical thinking, and winning habits.

---

## Selection Philosophy

We curate resources that help players think better—not just play better.

> *"The skills that make a great Habib Punja player—pattern recognition, risk assessment, opponent modeling—are the same skills that drive success in business, investing, and life."*

---

## Selection Criteria

### Inclusion Requirements

All resources must meet these standards:

| Criterion | Description |
|-----------|-------------|
| **Credibility** | From recognized institutions or established experts |
| **Applicability** | Directly relevant to strategy, decision-making, or risk control |
| **Accessibility** | Clear writing/speaking style appropriate for players |
| **Integrity** | Avoids sensationalism or pseudo-science |
| **Practicality** | Supports learning through play, not just theory |

### Exclusion Criteria

We do not include:
- Unverified or anonymous sources
- Content promoting harmful behavior
- Purely promotional material
- Outdated information (when currency matters)
- Paywalled content without free alternatives
