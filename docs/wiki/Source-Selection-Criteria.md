# Source Selection Criteria

This project includes a curated set of public references supporting strategic mindset, critical thinking, and winning habits.

Selection criteria (high-level):
- credible institutions or recognized experts
- directly applicable to strategy, decision-making, and risk control
- accessible writing/speaking style for players
- avoids sensationalism or pseudo-science
- supports practical learning through play
